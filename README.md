## 开发说明

主要使用 FastAPI + SQLModel 搭建基础骨架，包含示例路由、数据库模型、CRUD、测试与 Dockerfile。

快速运行（推荐在虚拟环境中执行）：

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

运行测试：

```bash
pytest -q
```
